# Proyecto Unidad IV - Despliegue Web con Contenedores y IaC

Este proyecto implementa una aplicación web desplegada mediante contenedores y se define infraestructura como código (IaC) usando Terraform.

## Contenido del Repositorio
- **app/** → Código fuente en Node.js
- **Dockerfile** → Imagen Docker de la aplicación
- **terraform/** → Scripts IaC de VPC, ECS, ALB, RDS y DNS
- **.github/workflows/** → Pipeline CI/CD (GitHub Actions)
